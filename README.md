# Prontuário Odontológico

Sistema local para consultório odontológico.

## Instalação simples

Execute:

```text
INSTALAR_PRONTUARIO.bat
```

## Abertura do sistema

Após instalado, use o atalho criado na área de trabalho:

```text
Prontuario Odontologico
```

## Primeiro acesso

```text
Email: admin@odonto.com
Senha: 123456
```

Troque a senha após o primeiro acesso.

## Backup

Use o atalho:

```text
Fazer Backup do Prontuario
```

## Requisito

Docker Desktop instalado e em execução.

## Atalhos

O instalador cria atalhos na área de trabalho com ícones próprios para abrir o sistema, fazer backup e encerrar o Prontuário.
